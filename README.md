# Connect Four (Python Terminal Game)

A lightweight, fully functional, two-player Connect Four game that runs directly in your terminal or command prompt. Built entirely with Python's standard library.

---

## 🌟 Features

* **Two-Player Hotseat:** Play locally against a friend using the same keyboard.
* **Smart Board Gravity:** Pieces automatically "drop" to the lowest available slot in a chosen column.
* **Instant Win Detection:** Accurately calculates horizontal, vertical, and diagonal (both positive and negative slopes) wins.
* **Robust Input Handling:** Prevents crashes from accidental typos (like typing letters instead of numbers) and stops players from dropping pieces into full columns.

---

## 🚀 Prerequisites

This game uses only built-in Python modules. You do not need to install any external packages (like `pygame` or `numpy`). 

* **Python 3.x** must be installed on your computer.

---

## 🎮 How to Run

1. Save the game code into a file named `connect_four.py`.
2. Open your terminal or command prompt.
3. Navigate to the folder where you saved the file.
4. Run the following command:

   ```bash
   python connect_four.py
